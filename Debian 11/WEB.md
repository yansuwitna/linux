# WEB SERVER DENGAN APACHE dan PHP

**1. Instalasi Web Server**
```console
apt-get install apache2 php
```

**Membuat Folder**
```console
mkdir /var/www/html/utama
mkdir /var/www/html/osis
```

**Membuat File html**
```console
nano /var/www/html/utama/index.php
```

isikan   

```console
Website Utama
```
